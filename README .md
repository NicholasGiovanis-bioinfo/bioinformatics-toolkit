#  Bioinformatics Toolkit (Python)

\> A beginner-friendly bioinformatics toolkit for DNA analysis and
protein translation.

This project is a collection of bioinformatics tools implemented in
Python. It focuses on DNA sequence analysis, mutation detection, and
protein translation.

\-\--

##  Purpose

This project was built as an introduction to bioinformatics, combining
programming and biological concepts to analyze DNA sequences and
simulate protein translation.

\-\--

##  Features

\- DNA sequence validation - GC content calculation - Mutation detection
between two sequences - DNA → mRNA → amino acid translation - Stop codon
detection

\-\--

##  Example Usage

\### Input:

\`\`\` DNA Sequence: ATGTTAATT \`\`\`

\### Output:

\`\`\` mRNA sequence: AUGUUAAUU Translated into amino acids:
\[\'Methionine\', \'Leucine\', \...\] \`\`\`
